發表於 2020-03-13 | 分類於 JavaScript |

陣列(Array)

當我們需要處理大量相同屬性的物件時，使用變數去宣告的話會造成變數過多，影響到資料處理的效率，因此我們可以將相同屬性的東西以陣列的方式去表示。例如：農場中有主人、2隻狗、15隻雞、3個玉米田、4個花椰菜田跟9個稻草人時，我們可以用下圖的方式去表示
